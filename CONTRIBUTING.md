# Contributing

## Setup

```bash
# Clone the repository
$ git clone git@github.com:umijs/tnf.git
$ cd tnf
# Install dependencies
$ pnpm install
# Install Playwright browsers
$ pnpm playwright install chromium
```

## Development

```bash
# Start the development server
$ pnpm dev
```

## Build

```bash
# Build the project
$ pnpm build
```

## Test

```bash
# Run all tests in watch mode
$ pnpm test
# Run all tests
$ pnpm test --run
# Run e2e tests
$ pnpm test:e2e
```
